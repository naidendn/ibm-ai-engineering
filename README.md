# IBM AI Engineering Course Exercises

This repository contains exercises, sample code, and projects completed as part of the **IBM AI Engineering Professional Certificate** program.  

It serves as a hands-on collection of work demonstrating key concepts and practical applications in:

- Data analysis and visualization using **pandas** and **matplotlib**
- Machine learning models such as **linear regression** and **classification**
- Model evaluation and validation techniques
- Python programming best practices for AI workflows

The materials in this repository are intended for educational purposes, helping learners understand and apply AI and machine learning concepts in real-world scenarios.
